# 釣りクエスト ～海釣り編～

> **注意（2026-07-10）:** 以下は初期MVP時点のスナップショットで、現行V2の機能・検証状況を反映していません。現在地は `docs/30_v2_expansion_overview.md`、発売前監査と次タスクは `docs/45_release_readiness_code_review.md` を参照してください。製品版READMEへの全面更新はリリース前タスクです。

Godot 4.7向けのMVPプロトタイプ兼、要件定義・設計資料一式です。

> **ゲームの核**  
> 釣る → 持ち帰る → 売る／調理する → 食べて成長する → 装備を強化する → ぬしに挑む

## すぐに動かす（Mac）

1. Godot 4.7 Standard版を公式サイトから入手します。macOS版はApple Silicon／Intel共通のUniversalビルドです。  
   https://godotengine.org/download/archive/4.7-stable/
2. Godotを起動し、プロジェクトマネージャーでこのフォルダの `project.godot` を読み込みます。
3. 右上のプロジェクト再生ボタン、または `F5` で実行します。
4. macOSでアプリを開けない場合は、FinderでGodotをControlクリックして「開く」を選択します。

`tools/open_on_mac.command` をダブルクリックして起動する方法も用意しています。初回のみ、ターミナルで次を実行してください。

```zsh
chmod +x tools/open_on_mac.command
```

## 操作

| 場面 | 操作 |
|---|---|
| メニュー | マウス、Enter |
| 仕掛けを投げる | 画面ボタン、Enter |
| アワセる | 画面ボタン、E、Enter |
| リールを巻く | 画面ボタン長押し、Space長押し |
| 糸を出す | 画面ボタン長押し、Shift長押し |

水中ファイトでは、テンションを安全域に保ったまま魚の体力を削ります。魚が突進・潜水している間は糸を出し、魚が休んだら巻くのが基本です。

## このプロトタイプで動くもの

- タイトル画面とセーブ初期化
- 港を中心とした画面遷移
- 通常魚5種＋港のぬし1種
- 水中で魚の動きが見える釣りファイト
- テンション、魚体力、プレイヤー体力、距離、水深
- 魚の売却
- 料理、食経験値、レベル1～10
- 初回料理ボーナス
- 次の釣行だけ有効な食事効果
- 竿3種の購入・装備
- 魚図鑑、最大サイズ、料理記録
- JSONセーブ／ロード

## 検証状況

同梱GDScriptは `gdformat` で整形し、`gdlint` で全ファイルの構文解析・静的チェックを通しています。この作業環境ではGodot本体を起動できなかったため、Mac上で `tools/validate_project.sh` を実行してエンジンによる最終確認を行ってください。詳しくは `VALIDATION.md` を参照してください。

## 最初に触る場所

ゲームバランスは `src/autoload/game_data.gd` に集約しています。

- `FISH`: 魚の体力、引き、速度、価格、食経験値
- `RECIPES`: 料理倍率、解放レベル、食事効果
- `RODS`: 価格と装備補正

レベル計算とセーブは `src/autoload/player_progress.gd`、釣りの挙動は `src/core/fishing_simulator.gd` にあります。
