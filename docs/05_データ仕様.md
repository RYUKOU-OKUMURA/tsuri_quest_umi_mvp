# データ仕様

## 1. Fish

| キー | 型 | 内容 |
|---|---|---|
| id | String | 内部ID |
| name | String | 表示名 |
| rarity | String | レア度 |
| min_level | int | 通常抽選または挑戦解放レベル |
| boss | bool | ぬし判定 |
| weight | float | 通常魚抽選の基本ウェイト |
| size_min/max | float | サイズ範囲cm |
| sell_price | int | 1匹の売価 |
| food_exp | int | 料理前の基礎食経験値 |
| stamina | float | 魚体力 |
| power | float | テンションへの圧力 |
| speed | float | 距離変化・画面移動補正 |
| start_distance | float | ファイト開始距離m |
| start_depth | float | 初期水深m |
| color | String | フォールバック表示色 |
| habitat | String | 図鑑説明 |
| behavior | String | 行動説明 |

## 2. FishingSpot

| キー | 型 | 内容 |
|---|---|---|
| id | String | 内部ID |
| name | String | 表示名 |
| unlock_level | int | 釣り場解放レベル |
| required_boat_rank | int | 出航に必要な船ランク。0は船不要 |
| depth_range | Vector2 または Array[float] | 水深目安m |
| description | String | 選択画面用説明 |
| common_modifier | float | 明示補正がない候補魚の基本補正 |
| fish_weight_modifiers | Dictionary | 魚IDごとの出現weight補正 |
| featured_fish | Array[String] | 選択画面で見せる代表魚 |
| recommended_baits | Array[String] | おすすめエサ |
| boss_spot | bool | ぬし専用ポイントか |

釣り場補正は魚の `weight` に掛ける。Lv.1〜2帯の魚は多くの通常ポイントで候補に残し、高レベル魚は対応ポイントでも低確率に抑える。`required_boat_rank` は出航可否だけに使い、出現weightには直接影響しない。出現抽選の詳細は `docs/15_fishing_spot_encounter_spec.md`、船アクセスの詳細は `docs/17_boat_access_spec.md` を正とする。

## 3. Recipe

| キー | 型 | 内容 |
|---|---|---|
| id | String | 内部ID |
| name | String | 調理法名 |
| unlock_level | int | 解放レベル |
| exp_multiplier | float | 食経験値倍率 |
| allowed_fish | Array[String] | 対応魚種 |
| description | String | 説明 |
| buff_stat | String | 効果対象 |
| buff_value | float | 効果量 |
| buff_text | String | UI表示 |

## 4. Rod

| キー | 型 | 内容 |
|---|---|---|
| id | String | 内部ID |
| name | String | 表示名 |
| price | int | 購入価格 |
| reel_multiplier | float | 巻力倍率 |
| line_limit_bonus | float | 切断限界加算 |
| technique_bonus | int | 技量加算 |
| description | String | 説明 |

## 5. Boat

| キー | 型 | 内容 |
|---|---|---|
| id | String | 内部ID |
| name | String | 表示名 |
| short_name | String | UI短縮名 |
| rank | int | 出航可能範囲を表す船ランク |
| price | int | 購入価格 |
| description | String | 説明 |
| access_text | String | 購入後に広がる出航範囲 |

船は装備切り替えを持たない恒久アンロックとして扱う。所持している船の最大 `rank` が、釣り場の `required_boat_rank` 以上なら出航できる。

## 6. 食事効果

| buff_stat | 適用 |
|---|---|
| max_energy | 最大体力を割合増加 |
| bite_window | アワセ時間を秒加算 |
| safe_range | 安全域の上下を拡張 |
| energy_regen | 体力回復量を割合増加 |
| reel_power | 巻力を割合増加 |

## 7. ID命名規則

- 小文字の英数字とアンダースコアを使用
- 表示名から独立させる
- セーブ後は既存IDを変更しない
- 削除する場合はセーブ移行処理を用意する
