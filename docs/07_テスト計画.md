# テスト計画

## 1. スモークテスト

| ID | 手順 | 期待結果 |
|---|---|---|
| ST-01 | プロジェクトを起動 | タイトルが表示される |
| ST-02 | 新規開始→港→釣り | 水中画面が表示される |
| ST-03 | キャストしてアワセをしない | 逃走結果になる |
| ST-04 | アワセ後に巻き続ける | 高テンションでライン切れになる |
| ST-05 | 糸を出し続ける | 低テンションで針外れになる |
| ST-06 | 安全域で操作し釣り上げる | 魚が在庫と図鑑に追加される |
| ST-07 | 市場で売る | 在庫減少、所持金増加 |
| ST-08 | 調理して食べる | 在庫減少、経験値増加、バフ設定 |
| ST-09 | 次の釣行へ入る | バフ適用後、保留バフが消える |
| ST-10 | 竿を購入 | 所持金減少、装備更新 |
| ST-11 | 船を購入 | 所持金減少、所持船に追加、出航可能範囲が広がる |
| ST-12 | 再起動 | 進行が復元される |
| ST-13 | 釣りでREADY〜BITEまで進める | 魚名、魚画像、No.、レアリティ、推定サイズ、魚固有の好むエサが表示されず、右カードは未確認の魚影カードになる |
| ST-14 | BITEでアワセず逃走する | 魚種が最後まで未確認のまま逃走結果になる |
| ST-15 | BITEでアワセてFIGHTへ入る | 水中ファイト開始時に初めて魚種カードと水中の魚が表示される |
| ST-16 | READYで下部HUDの港へ戻るを押す | 確認なしで港へ戻る |
| ST-17 | CASTING〜BITEで下部HUDの港へ戻るを押す | 「釣りを中断して港へ戻りますか？」の確認が表示される |
| ST-18 | FIGHTで下部HUDの港へ戻るを押す | 「ファイトを中断すると魚は逃げます。港へ戻りますか？」の確認が表示される |
| ST-19 | 帰港確認で続けるを押す | 確認が閉じ、釣り状態が維持される |
| ST-20 | 帰港確認で港へ戻るを押す | 釣果を記録せず港へ戻る |
| ST-21 | `Esc` または `-` を押す | 下部HUDの港へ戻ると同じ帰港導線になる |
| ST-22 | 港→釣り場選択へ進む | 解放済み釣り場カードと未解放ロックが表示される |
| ST-23 | 未解放釣り場を選ぶ | 釣り画面へ進まず、必要Lvが読める |
| ST-24 | 発見済みだが船不足の沖釣り場を選ぶ | 釣り画面へ進まず、必要な船が読める |
| ST-25 | 解放済みかつ船条件を満たした釣り場を選ぶ | 選択した釣り場情報を持って釣り画面へ進む |
| ST-26 | 高レベル釣り場でREADY〜BITEまで進める | 魚種は伏せたまま、選択中の釣り場と反応情報だけが表示される |

追加の機械的確認：

```zsh
godot --headless --path . res://tools/fishing_reveal_smoke.tscn
godot --headless --path . res://tools/fishing_harbor_return_smoke.tscn
```

このスモークは `prepare`、`CASTING`、`BITE` では魚種公開フラグが立たず、`hook()` 成功後だけ公開されることと、BITE逃走では未公開のままになることを確認する。
帰港スモークは、`READY` の即帰港、`CASTING` と `FIGHT` の確認文言、確認中の進行停止、`Esc` / `-` / `Enter` のキーボード導線を確認する。

## 2. 境界値テスト

- 所持魚0で調理・売却できない。
- 所持金が価格より1G少ないと購入できない。
- レベル10では経験値が増えてもオーバーフローしない。
- レベル未到達の釣り場ポイントを選択できない。
- 必要船ランク未到達の釣り場ポイントを選択できない。
- 必要船を所持していても、レベル未到達の釣り場ポイントは選択できない。
- レベル未到達の魚はどの釣り場でも抽選されない。
- ぬしは通常魚抽選に混ざらない。
- テンションが切断限界と同値のときは即切断せず、超えた時点で切れる。
- 魚体力0でも距離が残っていれば即捕獲されない。
- セーブに未知の魚IDがあっても状態画面がクラッシュしない。

## 3. バランステスト指標

各魚について10回以上記録する。釣り場ポイント導入後は、各釣り場で1万回程度の抽選サンプルも記録する。

- 捕獲成功率
- 平均ファイト時間
- ライン切れ率
- 針外れ率
- 平均残体力
- プレイヤーが「突進に気づいた」と答えた割合
- 釣り場別の魚種出現率
- Lv.1〜2魚が各通常ポイントに混ざる割合
- レア魚、大物魚の1匹あたり出現率
- 目的魚を狙うための釣り場選択がプレイヤーに伝わった割合

初期目標：

| 対象 | 成功率 | 平均時間 |
|---|---:|---:|
| アジ Lv.1 | 70～90% | 15～30秒 |
| メジナ Lv.1 | 45～70% | 25～45秒 |
| サバ Lv.3 | 40～65% | 30～55秒 |
| ぬし Lv.5入門竿 | 10～30% | 60～120秒 |
| ぬし Lv.8磯竿 | 50～75% | 45～90秒 |

出現率の初期目標：

| 対象 | 目安 |
|---|---:|
| Lv.1〜2魚の合計 | 高レベル通常ポイントでも40%以上 |
| アンコモン1種 | 対象ポイントで12〜20%以内 |
| レア中型1種 | 対象ポイントで6〜12%以内 |
| レア大型1種 | 対象ポイントで3〜8%以内 |
| 終盤大物1種 | 対象ポイントで1〜5%以内 |
