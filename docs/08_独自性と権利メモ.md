# 独自性と権利メモ

## 方針

ジャンルの基本構造を参考にしつつ、具体的な表現は独自にする。

## 本作固有の中心要素

- 釣った魚を料理して食べることがレベルアップ手段
- 魚を売るか食べるかの資源選択
- 料理の永久成長＋次回釣行バフ
- 魚の行動が見える水中ファイト
- 海の生態と料理文化を一つの成長体験として扱う

## 避けること

- 既存作品のタイトル、ロゴ、固有名詞の流用
- マップ、物語、人物、会話、魚グラフィックの模写
- 画面レイアウトをピクセル単位で再現すること
- 既存作品の音楽・効果音・スクリーンショットを製品へ同梱すること
- 宣伝文で公式の続編・関連作と誤認させること

## 制作ルール

- 最終アートはゼロから制作または利用許諾を明確にする。
- 使用素材の作者、ライセンス、入手元を台帳化する。
- 実在魚の写真をトレースする場合も写真ライセンスを確認する。
- 企画用モックアップは製品用素材と分離する。

この文書は制作上のリスク整理であり、個別案件の法的助言ではない。
