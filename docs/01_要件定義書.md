# 要件定義書 v0.1

## 1. 目的

『釣りクエスト ～海釣り編～』の中核体験を最小規模で実装し、以下を検証する。

- 水中の魚を見ながら操作する釣りが楽しいか
- 魚を「売る／食べる」選択が成立するか
- 料理によるレベルアップが、ぬし攻略への動機になるか

## 2. 対象ユーザー

- 釣りゲーム、収集ゲーム、RPGが好きなプレイヤー
- 複雑なシミュレーターより、分かりやすい操作と成長を好む層
- 10分程度の短いプレイでも進展を得たい層

## 3. スコープ

### 3.1 MVPに含める

- 港をハブとしたメニュー式進行
- 釣り場1つ
- 通常魚5種、ぬし1種
- キャスト、食いつき、アワセ、水中ファイト、釣果
- テンション管理
- 魚売却
- 料理5種
- 食経験値、レベル1～10
- 次の釣行のみ有効な食事効果
- 竿3種
- 魚図鑑と料理記録
- 自動セーブ

### 3.2 MVPに含めない

- オープンフィールド探索
- オンライン要素
- リアルタイム天候、潮汐
- 料理アクションゲーム
- 複雑なクラフト
- NPC会話、クエストチェーン
- 課金、広告

## 4. 機能要件

### FR-001 タイトル

- タイトル、サブタイトル、コンセプト文を表示する。
- セーブがある場合は「つづきから」を選べる。
- 「最初から」で既存進行を確認後に初期化できる。

### FR-010 港

- 釣り場、調理場、魚市場、釣具店、ステータスへ遷移できる。
- レベル、所持金、装備、所持魚数、食事効果を確認できる。

### FR-020 釣り準備

- 通常魚またはぬしを選択できる。
- ぬしはレベル5未満では選択不可とする。
- 釣行開始時、保留中の食事効果を1回だけ適用・消費する。

### FR-021 食いつき

- キャスト後、待機・接近・食いつきの状態を経る。
- 食いつき中にアワセ操作を行うとファイトへ移行する。
- 制限時間を過ぎると逃走となる。

### FR-022 水中ファイト

- 魚を水中に表示する。
- 魚は突進、潜水、方向転換、休息の行動を取る。
- プレイヤーは巻く、糸を出すを選べる。
- テンションが上限を超えるとライン切れになる。
- テンションが低すぎる状態が続くと針外れになる。
- 安全域で巻くと魚体力を効率よく減らせる。
- 魚体力が十分低く、距離が0に近づくと釣り上げる。

### FR-023 釣果

- 魚名、サイズを表示する。
- クーラーボックスに1匹追加する。
- 釣果数と最大サイズを図鑑へ記録する。

### FR-030 魚市場

- 所持魚と個数、単価を表示する。
- 1匹売却、全数売却を選べる。
- 売却後に所持金と在庫を即時更新する。

### FR-040 調理

- 所持魚を選び、現在レベルで解放済みの料理を表示する。
- 魚を1匹消費し、食経験値を得る。
- 魚種×調理法の組み合わせを初めて食べた場合、経験値を2倍にする。
- 最新の料理による食事効果を「次の釣行」に設定する。

### FR-050 成長

- レベルは1～10とする。
- レベルアップで最大体力、巻力、技量、集中力を上げる。
- レベル5でぬし挑戦を解放する。
- レベル上昇は操作を自動化せず、許容範囲と効率を改善する。

### FR-060 装備

- 竿を購入、装備できる。
- 竿は巻力、切断限界、技量を補正する。
- 所持している竿への装備変更は無料とする。

### FR-070 図鑑・状態確認

- 魚の発見数、釣果数、最大サイズを表示する。
- 所持魚、所持竿、食べた料理を表示する。

### FR-080 セーブ

- 進行状況をJSONで `user://tsuri_quest_save.json` に保存する。
- 捕獲、売却、料理、購入、釣行開始時に自動保存する。
- 壊れたセーブデータではクラッシュせず初期値を使用する。

## 5. 非機能要件

### NFR-001 性能

- 1280×720で60fpsを目標とする。
- MVPでは同時に描画する大型魚を1匹に限定する。
- 水中背景は軽量な2D描画とする。

### NFR-002 操作性

- マウスのみでも基本ループを完結できる。
- 釣り操作はキーボードでも実行できる。
- 押しっぱなし操作は画面上で明示する。

### NFR-003 保守性

- 魚、料理、竿の値は `game_data.gd` へ集約する。
- 画面、進行、釣りシミュレーションを分離する。
- 画面同士はMainルーターを介して遷移する。

### NFR-004 可搬性

- Godot Standard版だけで編集・実行できる。
- 外部プラグイン、外部フォント、ネイティブ拡張を必須にしない。

## 6. 受入基準

| ID | 受入条件 |
|---|---|
| AC-01 | 新規開始後、3クリック以内に釣り場へ移動できる |
| AC-02 | キャストから釣果または逃走まで停止せず進行する |
| AC-03 | 魚を釣ると所持数と図鑑が増える |
| AC-04 | 売却すると在庫が減り、所持金が増える |
| AC-05 | 調理すると在庫が減り、経験値と食事効果が増える |
| AC-06 | レベルアップ後、釣り能力値が上昇する |
| AC-07 | レベル5未満ではぬしを選べず、レベル5以上で選べる |
| AC-08 | ゲーム再起動後も進行が復元される |
